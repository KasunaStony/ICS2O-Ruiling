Introduction

A new generation firewall (NGFW) is a hardware or software -based network security system. It can fully address the threat of high-performance firewall. NGFW can provide users with an effective integration of application-layer security through insight into the network traffic of users, applications and content with the new high-performance single-path heterogeneous parallel processing engine, it helps users to safely conduct business and simplify the user network security architecture.
